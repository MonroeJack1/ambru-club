# Ambru Club Landing Page

A modern responsive landing page for **Ambru Club**, built with Next.js App Router, React, Tailwind CSS and shadcn/ui-style components.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Main files

- `src/app/page.tsx` - the landing page sections and Romanian copy
- `src/app/globals.css` - theme tokens, amber/navy palette and subtle visual effects
- `src/components/ui/*` - shadcn/ui-style Button, Card, Badge, Input and Accordion components

## Notes

The email signup forms are presentational and accessible. Connect them to your newsletter, CRM or API by adding a form action or a Server Action.
